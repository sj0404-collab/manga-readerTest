📖 КРАТКОЕ ОПИСАНИЕ
═══════════════════════════════════════════════════════════════════════

Manga Talk Reader - это инновационное Android приложение для чтения манги
с интегрированной технологией OCR (распознавание текста) и TTS (озвучка).

✨ КЛЮЧЕВЫЕ ОСОБЕННОСТИ:

  ✓ Мультиязычный OCR (РУ, ЯП, АНГ)
  ✓ Озвучивание текста (TTS с кастомизацией)
  ✓ Поддержка архивов (ZIP, CBZ, CBR)
  ✓ Управление библиотекой
  ✓ Material Design 3 UI
  ✓ MVVM архитектура
  ✓ Jetpack Compose UI
  ✓ Room Database
  ✓ Hilt Dependency Injection

🚀 БЫСТРЫЙ СТАРТ (5 МИНУТ)
═══════════════════════════════════════════════════════════════════════

1. ТРЕБОВАНИЯ:
   ✓ Android Studio 2024.1.1+
   ✓ Android SDK 34+
   ✓ Java JDK 17+
   ✓ Интернет для загрузки зависимостей

2. КЛОНИРОВАНИЕ:
   git clone <repository-url>
   cd manga-talk-reader

3. ОТКРЫТИЕ В ANDROID STUDIO:
   - File → Open → выбрать корневую папку
   - Дождаться синхронизации Gradle

4. СБОРКА DEBUG APK:
   ./gradlew assembleDebug
   (результат: app/build/outputs/apk/debug/app-debug.apk)

5. УСТАНОВКА И ЗАПУСК:
   ./gradlew installDebug
   adb shell am start -n com.mytech.mangatalkreader/.MainActivity

🔑 КЛЮЧЕВЫЕ КОМПОНЕНТЫ
═══════════════════════════════════════════════════════════════════════

DATA LAYER:
  - Room Database (SQLite) с 6 таблицами
  - MangaRepository для бизнес-логики
  - Полная поддержка Coroutines

DOMAIN LAYER:
  - OcrService (ML Kit Text Recognition)
  - TtsService (Android TextToSpeech)
  - ArchiveService (ZIP/CBZ обработка)

UI LAYER:
  - 6 Compose экранов
  - MVVM с StateFlow
  - Material Design 3

📦 СБОРКА RELEASE APK
═══════════════════════════════════════════════════════════════════════

1. Debug APK (для разработки):
   ./gradlew assembleDebug
   → app/build/outputs/apk/debug/app-debug.apk (~45 MB)

2. Release APK (для публикации):
   ./gradlew assembleRelease
   → Требует подписания (смотри BUILD_GUIDE.md)

3. Bundle для Google Play:
   ./gradlew bundleRelease
   → app/build/outputs/bundle/release/app-release.aab

❓ ЧАСТЫЕ ВОПРОСЫ
═══════════════════════════════════════════════════════════════════════

Q: Как импортировать мангу?
A: Нажмите FAB кнопку в Library → выберите Import Archive

Q: OCR не распознаёт текст?
A: Убедитесь, что изображение достаточно чёткое

Q: TTS не работает?
A: Проверьте, что язык установлен в системе

Q: Как опубликовать на Google Play?
A: Смотрите BUILD_GUIDE.md раздел "Release APK"
